# 用户说明书：Texture Atlas 编辑器

## 环境

- 现代 Chromium / Edge / Firefox（支持 ES 模块与 Canvas）。
- 无需安装即可使用开发服务器；正式发布为静态文件，可用任意静态服务器打开。

## 启动（开发）

在 `frontend-vue` 目录执行 `npm run dev` 或使用 `dev.py`，浏览器访问终端提示的本地 URL。

## 功能概要

0. **帮助**：**支持的格式…** 首段说明本应用不采用专有自定义格式的原因；其余为导入 / 导出 / 逆向的文件类型对照；**名词概念…** 解释下列用语（与界面一致）。
   - **图集（Atlas）**：主要指**描述文件**（如 JSON、XML），记录布局信息与**像素图路径**等，再与对应像素图配合。
   - **像素图**：保存栅格像素的文件，如 PNG、BMP、TGA、TIFF 等（本应用导出合并结果为 PNG）。
   - **离散像素图**：多张独立像素图构成的**列表**，可导入后参与打包。
   - **打包（Pack）**：将离散像素图排布成**打包后的像素图**，并生成描述数据。
   - **打包后的像素图**：Pack 得到的合并位图（可多页）。
   - **拆包（Unpack）**：由描述文件 + 打包后的像素图还原为离散像素图；**导入图集**与**逆向拆分**均属此类，前者进列表、后者下载文件。
   - **关系**：离散像素图 → Pack → 打包后的像素图 + 描述文件；后者 → Unpack → 离散像素图。
1. **在图片列表中右键「导入图片」**，或拖入光栅图：加入左侧列表（**菜单与工具栏不再提供「导入图片」**）。
2. 单击列表项查看**属性**（名称、宽高）。在**图片列表**空白或条目上**右键**：**导入图片**、**删除**（针对当前条目）、**全部删除**（会确认）。
3. **列表双击**某张图：弹出**原图全尺寸预览**（非画布平移）。
4. **图集**（主区预览面板，原「画布」区域）：未贴图区域为**透明**；导出 PNG 亦为透明底。工具条 **「辅助线」** 下可分别勾选：**网格**、**单张上限**（紫）、**输出边界**（橙）、**图块描边**（青）；另有**线宽 (px)**（纹理像素，1～128；默认约为单张 **max(W,H) 的 1/256**，改「单张最大宽/高」会重算默认线宽；若仅再次打包且上限数字未被修正，则保留你手调的线宽）与**网格步长**。**金色**为**选中**（独立显示，不受上述四项影响）。图例随勾选项显示。标题栏含页码与单张上限；多页可用 **「全部总览」**。**单击**选中图块。滚轮缩放；中键或空格+左键平移。
5. **工具栏（三组）**：① **导入图集** + **导出图集**（均先弹出**格式选择**，再选文件或开始下载）；② **运行打包**；③ **逆向拆分**（先选格式再选 JSON+PNG，**下载**子图，不加入列表）。**导入图集**与**逆向拆分**用途不同：前者把子图**加入列表**可再编辑，后者仅**导出**多张 PNG。
6. 在 **打包图集**面板可 **运行打包**并设置单张上限与算法；**导入图集**、**导出图集**、**逆向拆分**仅通过菜单「文件」或工具栏（均先选格式再选文件）。超出单张范围则拆成**多页图集**。
7. **菜单「文件」**：**新建**（清空列表与图集预览；有内容时先确认）、**导入图集**、**导出图集**、**逆向拆分**（无「导入图片」项）。
8. **导出图集**：可选 **PNG+JSON**、**仅 PNG**、**仅 JSON**（多页时各页一套文件）。
9. **导入图集**：选「本应用 v1」后依次选 **JSON** 与 **PNG**，子图进入列表。**逆向拆分**：同样选文件对，子图**下载**到本机。
10. **主菜单**：**语言(L)**（简体中文 / English）、**主题(T)**（浅色 / 深色 / 跟随系统）；**设置(S)** 下暂无其它项（占位）。语言与主题偏好保存在浏览器本地存储。

## 常见问题

- **无法导出**：请先成功运行打包且列表非空。
- **逆向失败**：确认 JSON 与 PNG 为同一导出批次且未改尺寸。
