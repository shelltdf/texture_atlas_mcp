# 接口设计（程序间数据：导出清单）

本文件描述**导出/逆向**所用的 JSON 清单形态，**不含** GUI 操作说明。

## Atlas Manifest（JSON）

- **version**：数字，当前为 `1`。
- **width** / **height**：整数，图集画布像素尺寸。
- **sprites**：数组，每项包含：
  - `id`：字符串，与编辑器内条目 id 一致。
  - `name`：字符串，原始文件名或展示名。
  - `x`, `y`, `w`, `h`：整数，在图集中的像素矩形（左上原点）。

## 传输

- 文件交换：用户磁盘上的 `.json` 与 `.png` 配对。
- MIME：`application/json` 与 `image/png`。

## 错误语义（逆向）

- JSON 解析失败：提示格式错误。
- 图片尺寸与清单 `width`/`height` 不一致：提示可能文件不匹配或已缩放。
