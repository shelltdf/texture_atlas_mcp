# 详细设计：Texture Atlas 编辑器

## 用例

1. **导入图片**：用户通过菜单或工具栏触发文件选择，多选；加入列表并异步生成缩略图。
2. **选择图片**：列表单击；属性区显示文件名、像素宽高、内存引用状态。
3. **设置单张上限并打包**：在算法选择之上输入「单张最大边长」；再选择 `grid` / `rows` / `skyline`，点击运行。超出单张上限时拆成多页结果，可切换预览。
4. **导出发布**：每页生成一对清单 + PNG；多页时使用序号文件名。
5. **逆向**：用户选择 `atlas.json` 与对应 `atlas.png`，解析后按 `sprites[]` 裁剪，逐张下载或打包提示。

## 状态与交互

- 未导入图片时：画布显示占位提示；打包按钮可禁用或提示无数据。
- 打包失败（如单图过大超出上限）：状态栏与对话框提示。

## 算法行为（逻辑层）

- **Grid**：固定列数或自动平方根列数，按顺序单元放置，画布扩展为覆盖所有单元。
- **Rows**：按行累积宽度，行高取该行最大高度，换行贪心。
- **Skyline**：底部天际线启发式放置，尽量降低画布浪费（实现可为简化版）。

具体边界与默认值以 `02-physical/texture-atlas-editor-spa/spec.md` 为准。
